## Plans
- Subscription: $39.99/month
- Pay-as-you-go (tokenized)

## Comparison

| Feature | Subscription ($39.99/mo) | Pay-as-you-go (tokens) |
| --- | --- | --- |
| Asset classes | All | Single |
| Technical analysis | Intraday/weekly tops & bottoms | On-demand, pay as you go |
| News feed | Advanced | Basic |
| AI Q&A | Unlimited | First 50/week free, then 20 tokens each |
| Top/bottom notifications | Unlimited | 100 tokens each |
| Support | Priority | Standard |
| Daily Q&A cap | None | 10/day |

## FAQs
▶ Switch billing: PAYG→Subscription takes effect immediately; Subscription→PAYG after cycle ends

▶ Token expiry: none

▶ Balance & usage: dashboard shows balance, history, and details

▶ Refunds: no refund after cycle start; unused tokens refundable within 7 days
